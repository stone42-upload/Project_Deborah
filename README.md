# Deborah-Corp - Cours de Mathématiques

Site web pour **Deborah-Corp**, organisme de cours particuliers de mathématiques.

## Stack Technique

- **Framework:** Next.js 15+ (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4.0
- **Animations:** Framer Motion
- **Forms:** React Hook Form + Zod
- **Icons:** Lucide React
- **Fonts:** Playfair Display + Inter

## Démarrage

```bash
# Installer les dépendances
npm install

# Lancer en développement
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```

## Déploiement

Compatible Vercel - Push sur GitHub et connecter le repo.
